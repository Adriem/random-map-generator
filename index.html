About Adriem's map generator

Hey, thank you for checking this page! This is a little project I started to experiment with web technologies while I tried some algorythms of random map generation. Make sure you have javascript enabled on this page for this to work.

This isn't intended to be the most efficient map generator nor the most flexible one. I'm also not trying to teach anything to anybody. This is just the result of some research and experimentation by my own. Please, keep this in mind, specially if you are going to use the code, which is available on GitHub.

That said, I really hope you enjoy playing around with this little generator and find it useful! If you have any suggestions, comments or just want to tell me how wonderful this project is, feel free to email me, I'll reply as soon as I can. You can check some personal information in contact page.

See you!
